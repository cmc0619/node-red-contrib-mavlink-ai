# Examples

Planned Node-RED example flows:

```text
01 UDP heartbeat listener
02 UDP routed multi-vehicle listener
03 Build and send heartbeat
04 Arm/disarm command
05 Request autopilot version
06 Download mission
07 Filter GLOBAL_POSITION_INT
08 Raw packet debug
09 Serial connection example
```

Example flows should be exported as JSON files once the runtime implementation exists.

Keep examples small and focused. A good example should demonstrate one concept, not become a full ground-control-station hairball.
